字符串方法

JavaScript 中的字符串是不可变的，但提供了许多方法来操作字符串。

  • charAt(index)：返回指定索引处的字符。
  • concat(string2, string3, ..., stringN)：连接两个或多个字符串，并返回新的字符串。
  • includes(searchString, position)：判断一个字符串是否包含在另一个字符串中，根据情况返回 true 或 false。
  • indexOf(searchValue, fromIndex)：返回在字符串中首次找到的值的位置，如果未找到则返回 -1。
  • slice(beginIndex, endIndex)：提取字符串的某个部分，并返回一个新的字符串（不包含结束索引处的字符）。
  • split(separator, limit)：将一个字符串分割成字符串数组。
  • substring(start, end)：提取字符串中两个指定的索引号之间的字符。
  • toLowerCase()：将字符串转换为小写。
  • toUpperCase()：将字符串转换为大写。

数组方法

数组在 JavaScript 中是非常有用的数据结构，提供了许多内置方法来操作数组。

  • push(element1, ..., elementN)：向数组的末尾添加一个或多个元素，并返回新的长度。
  • pop()：移除数组的最后一个元素，并返回该元素的值。
  • shift()：移除数组的第一个元素，并返回该元素的值。
  • unshift(element1, ..., elementN)：向数组的开头添加一个或多个元素，并返回新的长度。
  • concat(array2, array3, ..., arrayN)：合并两个或多个数组。
  • slice(begin, end)：返回一个从开始到结束（不包括结束）选择的数组的一部分。
  • splice(start, deleteCount, item1, item2, ...)：通过删除或替换现有元素或者添加新元素来修改数组内容。
  • indexOf(searchElement, fromIndex)：返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回 -1。
  • forEach(function(currentValue, index, arr), thisArg)：对数组的每个元素执行一次提供的函数。

对象方法

JavaScript 中的对象是可变的键值对集合。

  • hasOwnProperty(prop)：判断对象自身是否有某个属性。
  • isPrototypeOf(object)：判断该对象是否为另一个对象的原型。
  • propertyIsEnumerable(prop)：判断对象自身的某个属性是否可枚举。
  • toLocaleString()：返回对象的本地字符串表示。
  • toString()：返回对象的字符串表示。
  • valueOf()：返回指定对象的原始值。

函数式方法

JavaScript 支持函数式编程范式，提供了一些高阶函数。

  • map(function(currentValue, index, arr), thisArg)：创建一个新数组，其结果是该数组中的每个元素是调用一次提供的函数后的返回值。
  • filter(function(currentValue, index, arr), thisArg)：创建一个新数组，其包含通过所提供函数实现的测试的所有元素。
  • reduce(function(accumulator, currentValue, currentIndex, array), initialValue)：对数组中的每个元素执行一个由您提供的 reducer 函数（升序执行），将其结果汇总为单个返回值。
  • forEach(function(currentValue, index, arr), thisArg)：对数组的每个元素执行一次提供的函数。
